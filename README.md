# Mini Kitaplık

Bu proje, React ile geliştirilmiş basit bir kitaplık uygulamasıdır.  

## Özellikler

- Kitap listesi görüntüleme
- Başlık veya yazar arama
- Kategori filtreleme
- Favorilere ekleme ve çıkarma
- LocalStorage ile verileri saklama
